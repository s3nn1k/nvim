# Установка Зависимостей

В системе должен быть установлен winget

## Утилиты

1. GCC - winget install MartinStorsjo.LLVM-MinGW.MSVCRT

2. Git - winget install Git.Git

3. RipGrep - winget install BurntSushi.ripgrep.MSVC

4. Терминал с поддержкой NerdFont - winget install Microsoft.WindowsTerminal

5. Npm - winget install OpenJS.NodeJS

6. winget install Chocolatey.Chocolatey

## Языки

1. Golang - winget install GoLang.Go

2. Python - winget install Python.Python.3.13 (Optional)

3. Typescript - npm install -g typescript (Optional)

## Lsp-сервера

1. Gopls - go install -v golang.org/x/tools/gopls@latest

2. Lua-language-server - choco install lua-language-server

3. Pyright - npm install -g pyright (Optional)

4. Typescript-language-server - npm install -g typescript-language-server (Optional)

## Форматтеры

1. Prettier - npm install -g prettier

2. Sql-formatter - npm install -g sql-formatter

3. Goimports - go install golang.org/x/tools/cmd/goimports@latest

4. Black - pip install black (Optional)

5. Stylua - choco install stylua (Optional)
